# Fitness Web Platform

A full-stack fitness web application that helps users track workouts, monitor progress, and discover nearby gyms.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Features
- User authentication
- Workout guidance
- Fitness progress tracking
- Gym locator using Maps API
